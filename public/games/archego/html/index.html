Instruction Explanation
  • INBOX: Pick up a value from the input tray
  • OUTBOX: Send the current value to the output tray
  • COPYTO: Copy the current value to a memory slot
  • COPYFROM: Pick up a value from a memory slot
  • ADD: Add a memory slot's value to your current value
  • SUB: Subtract a memory slot's value from your current value
  • BUMP +: Increase a memory slot's value by 1 and pick it up
  • BUMP -: Decrease a memory slot's value by 1 and pick it up
  • JUMP: Jump to a specific step in your program
  • JUMPIFZERO: Jump only if the current value is zero
  • JUMPIFNEG: Jump only if the current value is negative
